Aura is a Fiverr-inspired full-stack freelance marketplace platform designed for engineering tasks.
It connects Buyers, Solvers, and Admins in a complete workflow system where users can create, apply, assign, and complete tasks in a structured and secure environment.

The platform is built with modern web technologies and provides a scalable architecture with authentication, role-based access control, and real-time workflow tracking.

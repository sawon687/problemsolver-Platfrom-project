## 📖 Overview

Aura is a **Fiverr-inspired full-stack freelance marketplace platform** designed for engineering tasks. It is built entirely with **Next.js (Full Stack)**, meaning both frontend and backend logic are handled within the same Next.js application using API routes.

It connects **Buyers, Solvers, and Admins** in a complete workflow system where users can create, apply, assign, and complete tasks in a structured and secure environment.

The platform uses a modern, scalable architecture with authentication, role-based access control (RBAC), and real-time workflow tracking.

---

## ✨ Key Features

### 🔄 Complete Freelance Workflow

* Buyers create engineering tasks
* Solvers browse and apply for tasks
* Buyers review applications and assign solvers
* Solvers complete and submit deliverables
* Buyers review and approve work
* Full lifecycle tracking from start to finish
